# AI Prompts Library

A personal collection of AI prompts organized by category.

## Structure

```
ai-prompts/
├── coding/       # Programming, debugging, code review prompts
├── writing/      # Content creation, editing, copywriting
├── analysis/     # Data analysis, research, summarization
├── creative/     # Brainstorming, ideation, creative tasks
├── research/     # Information gathering, fact-checking
└── templates/    # Reusable prompt templates
```

## Usage

Each prompt is stored as a markdown file with:
- **Title**: Brief description
- **Prompt**: The actual prompt text
- **Variables**: Placeholders to customize (if any)
- **Notes**: Tips for best results

## Adding a New Prompt

1. Choose the appropriate category folder
2. Create a new `.md` file with a descriptive name
3. Use the template format below

## Prompt Template

```markdown
# Prompt Name

## Description
Brief explanation of what this prompt does.

## Prompt
Your prompt text here with {{variables}} in double braces.

## Variables
- `{{variable1}}`: Description of what to replace
- `{{variable2}}`: Description of what to replace

## Example
Example of the prompt filled in.

## Notes
- Any tips for getting better results
```
