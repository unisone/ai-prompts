# Competitor Analysis

## Description

Deep-dive competitor analysis that goes beyond surface features. Uncovers positioning, strategy, and exploitable gaps.

## When to Use

- Entering a market with established players
- Refining your differentiation
- Preparing competitive battle cards
- Pricing strategy decisions

## Prompt

```
Analyze competitor: [COMPANY NAME]
My company: [YOUR COMPANY/PRODUCT]

<analysis_framework>
## Company Overview
- Founded, funding, team size
- Mission/vision statement
- Recent news/announcements

## Product Analysis
### Core Offering
- Primary use case
- Key features (top 5-7)
- Tech stack (if known)
- Integrations

### Pricing
- Pricing model (per seat, usage, flat)
- Tiers and what's in each
- Enterprise/custom pricing signals
- Free tier limitations

### User Experience
- Onboarding flow
- Time to value
- Learning curve
- Mobile/API support

## Go-to-Market
- Primary customer segment
- Positioning statement
- Key marketing messages
- Sales motion (PLG, sales-led, hybrid)
- Content strategy themes

## Strengths
What do they do genuinely well? (Be honest)

## Weaknesses
Where do they fall short? (Specific, not generic)

## Customer Sentiment
- Common praise (from reviews/testimonials)
- Common complaints (from G2, Reddit, Twitter)
- Churn reasons (if findable)

## Strategic Assessment
- Where are they headed? (product roadmap signals)
- What would worry them about us?
- What should worry us about them?

## Opportunities
Specific gaps we can exploit:
1. [Gap] → [How we address it]
2. [Gap] → [How we address it]
3. [Gap] → [How we address it]
</analysis_framework>

<output_format>
- Use tables where appropriate
- Include direct quotes from reviews when relevant
- Rate confidence (high/medium/low) for speculative sections
- End with 3 actionable recommendations
</output_format>
```

## Why It Works

1. **Structure prevents bias**: Forces looking at strengths, not just weaknesses
2. **Customer sentiment**: Reviews reveal truth marketing hides
3. **Strategic framing**: "What would worry them" reframes your advantages
4. **Actionable output**: Ends with specific opportunities

## Quick Version

For rapid competitive scan:

```
Quick competitor scan: [COMPANY]
Tell me:
1. Who they serve (ICP)
2. Core value prop (one sentence)
3. Pricing model
4. Top 3 strengths
5. Top 3 weaknesses (from user reviews)
6. One thing we could do that they can't
```

## Notes

- Best with web search for current info
- Cross-reference G2, Capterra, Reddit for real user sentiment
- Update quarterly - competitors evolve
- Don't underestimate: if they have customers, they're doing something right
